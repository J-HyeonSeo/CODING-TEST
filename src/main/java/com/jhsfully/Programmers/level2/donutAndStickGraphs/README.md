### 도넛과 막대그래프 [LEVEL2]

### 유형 => 그래프

### 문제 풀이 방식
- 각각의 그래프는 특징을 가지고 있습니다.
- DONUT 모양은 최초로 재방문한 정점이 처음 정점과 같은 경우,
- STICK 모양은 더 이상 갈 길이 없는 경우,
- 8자 모양은 중점이 여러 개의 경로를 가지고 있습니다.
- 우선 정점으로부터 2가지 이상의 길을 뻗어나가고, 진입 지점이 없는 곳을 찾아 중점으로 지정하고,
- 이를 기점으로 각 그래프의 유형의 위의 조건대로 탐색하게 될 경우,
- 원하는 정답을 구해낼 수 있게됩니다.
